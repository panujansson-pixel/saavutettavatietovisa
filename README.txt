- index.html
- styles.css

- quiz_pack_300.json (valinnainen laajempi kysymyspaketti)
- README tässä kansiossa

Asennus GitHub Pages:
1. Luo uusi repo GitHubiin.
2. Lataa kaikki tiedostot (Upload files).
3. Commit & push.
4. Settings -> Pages -> Source: main (root) -> Save.
5. Odota julkaisua ja avaa URL.

Paikallinen testi:
- Avaa terminaali ja aja: python -m http.server 8000
- Avaa http://localhost:8000/index.html
